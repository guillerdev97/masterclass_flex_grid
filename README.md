## Setup

To run this project, install it locally using npm.

Clone the project

```bash
  git clone https://github.com/guillerdev97/masterclass_flex_grid.git
```

Go to the project directory

```bash
  cd masterclass_flex_grid
```

Start live server

```bash
   open live server
```
